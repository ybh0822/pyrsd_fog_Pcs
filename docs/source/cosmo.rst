Specifying the Cosmology
========================

The :mod:`pyRSD.rsd.cosmology` module allows users to specify the desired
cosmological parameters, and it also provides the functionality for users
to interface with the `CLASS <https://class-code.net>`_ CMB Boltzmann code.
